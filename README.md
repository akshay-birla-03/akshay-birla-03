**🏦 Global Bank Statement Analyzer (GBSA)** — *Fintech · DocBoyz*
Banks issue statements in wildly inconsistent formats, so extraction is manual and error-prone.
I built an automated parsing engine in **Python + Google Gemini LLM + OCR/regex** that extracts,
normalises and structures transactions from diverse global statement formats into standardized
**JSON**, backed by a cloud ingestion/storage pipeline on **Amazon S3**.

**💬 Fintech Chatbot** — *Fintech · DocBoyz*
An **LLM + RAG** conversational assistant that answers natural-language financial questions over
structured banking/transaction data — turning raw records into instant, grounded answers.

**🗄️ TIFR Archival Intelligence** — *Research / Government · Globe Creater × Tata Institute of Fundamental Research*
A digitisation and **archival-management** solution for large-scale institutional records —
improving document retrieval, organization, accessibility and long-term preservation of archival data.
